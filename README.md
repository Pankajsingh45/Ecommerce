# 🛒 Full Stack Ecommerce Web Application

A modern, fully responsive ecommerce web application with product listing, category filtering, admin dashboard, and secure checkout flow.

## 🔗 Live Demo
[View Live](https://ecommerce-git-main-pankajsingh45s-projects.vercel.app)

## 📁 GitHub Repository
[View Code](https://github.com/Pankajsingh45/Ecommerce)

## ⚙️ Tech Stack

| Layer | Technology |
|---|---|
| Frontend | Next.js, React.js, Tailwind CSS |
| Backend | Node.js, Express.js |
| Database | PostgreSQL, Prisma ORM |
| Payment | Razorpay (UI Integration) |

## ✨ Features

- 🛍️ Product listing with category-based filtering and sorting
- 🔍 Dynamic product rendering from backend APIs
- 🧑‍💼 Admin dashboard for managing products and coupon codes
- 🎟️ Coupon validation and discount system
- 💳 Checkout flow with Razorpay payment UI
- 📱 Fully responsive cross-device design

## 🚀 Getting Started

### Frontend
```bash
cd frontend
npm install
npm run dev
```

### Backend
```bash
cd backend
npm install
node index.js
```
